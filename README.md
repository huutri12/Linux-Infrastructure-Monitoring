# Linux Infrastructure & Automated Monitoring System

## 🎯 Project Goal
Xây dựng hạ tầng Web Server tự phục hồi (Auto-healing) và hệ thống giám sát thời gian thực trên Ubuntu Server.

## ✨ Key Features
- **LEMP Stack:** Triển khai Nginx, MySQL, PHP để vận hành Web.
- **Auto-Healing:** Script Bash tự động kiểm tra sức khỏe và khởi động lại dịch vụ qua Crontab.
- **Real-time Monitoring:** Giám sát thông số hệ thống (CPU, RAM, Disk) qua Prometheus & Grafana.

## 🛠 Technologies
- Ubuntu Server, Bash Scripting.
- [cite_start]Nginx, MySQL. [cite: 53]
- Prometheus, Grafana, Node Exporter.
